# SentinelAI

An AI-powered surveillance system built using Python and OpenCV.

## Features

- Motion Detection
- Face Detection
- Intruder Snapshot Capture
- Automatic Video Recording
- Timestamp Overlay
- Real-time Webcam Monitoring

## Technologies Used

- Python
- OpenCV
- Haar Cascade Classifier

## Project Structure

face_detection.py
motion_detection_v2.py
sentinel_motion_v3.py
intruder.py

## Future Improvements

- Email Alerts
- Edge AI Deployment
- Raspberry Pi Integration
